####################################     Instructions for Drawing Your Graph/Map     ###################################

# 1) Copy the path of the .pkl file and paste it into the 'file' parameter in the config.yml file;
# 2) Configure the file with what you want to do;
# 3) Run draw.py;
# 4) Check the drawing;
# 5) If you want to save, type 'y' at the prompt screen;
# 6) If you don't like the drawing, type 'n' and make another one!
